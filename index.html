About this app

This is an unofficial episode guide for Game of Thrones, built as a simple offline-ready web app. It includes every season and episode with short original summaries.

Images included are safe placeholder artwork (not screenshots). You can replace them by swapping files in assets/.

Tip: For offline install (PWA), open this using a local web server (not file://), then click “Install”.
